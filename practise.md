## Commandes de base

### Gestion des conteneurs
```bash
# Lancer un conteneur
docker run -d --name mon-conteneur nginx

# Voir les conteneurs en cours
docker ps

# Arrêter un conteneur
docker stop mon-conteneur

# Supprimer un conteneur
docker rm mon-conteneur
